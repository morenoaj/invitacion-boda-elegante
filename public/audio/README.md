# Carpeta de Audio

## Instrucciones

Coloca tu archivo de audio aquí con el nombre: **tu-amor-y-el-mio.mp3**

La canción que se reproducirá es "Tu amor y el mío" de Victorio Vergara.

### Pasos:
1. Descarga o ten listo el archivo de audio en formato MP3
2. Renómbralo a: `tu-amor-y-el-mio.mp3`
3. Coloca el archivo en esta carpeta: `/public/audio/`
4. La música comenzará a reproducirse automáticamente cuando los usuarios visiten la invitación

### Notas:
- El reproductor tiene controles de play/pause
- La música se repetirá en loop automáticamente
- El botón de control está en la esquina inferior derecha
